Simple Hospitality CRM

Angular | TypeScript | Firebase

Simple Hospitality CRM is a user-friendly Customer Relationship Management (CRM) system designed for property management, developed using Angular, TypeScript, and Firebase as the database. This system provides an efficient solution for managing various properties, bookings, and associated statistics.

Using Firebase Authentication, property owners can create access credentials for booking agents, enabling secure logins to the system. Once logged in, users are greeted with a clear and concise dashboard displaying key statistics, including property-specific income, booking volumes, and upcoming check-in and check-out dates for guests.

The core functionality of Simple Hospitality CRM is based on the CRUD (Create, Read, Update, Delete) principle. Users have the flexibility to create individual property listings, manage bookings by associating them with available properties, and update or delete records as needed. The system empowers property owners and booking agents to efficiently handle their hospitality business with a streamlined approach to data management.

Whether you're a property owner looking to manage bookings or a booking agent handling reservations, Simple Hospitality CRM simplifies the process, providing a comprehensive tool to organize, analyze, and optimize property-related data.

Join

JavaScript | HTML | CSS

"Join" is a project management tool inspired by the Kanban system, designed to streamline task organization and enhance collaboration.

Developed using JavaScript, HTML, and CSS, the tool leverages a remote storage system to store and manage data efficiently.

The primary features include task creation and management, the ability to add and manage contacts, and the allocation of specific tasks to individual contacts.

The tool's dashboard offers a user-friendly interface with drag-and-drop functionality, allowing users to organize tasks seamlessly.

This project aims to provide a practical and intuitive solution for effective project management, fostering productivity and collaboration.

El Pollo Loco

JavaScript | HTML | CSS

I employed an object-oriented approach (OOP) to create an engaging hit and run game called "EL Pollo Loco" Players navigate the character Pepe through various levels, collecting salsa bottles and coins along the way. The objective is to skillfully maneuver through the level, encountering adversaries that can be eliminated by throwing collected salsa bottles and executing well-timed jumps from above.

The game features a series of challenges, with the ultimate test awaiting players at the end of each level—the formidable Endboss, a malevolent chicken that must be defeated to progress. "EL Pollo Loco" aims to deliver a fun and interactive gaming experience, combining strategic thinking, quick reflexes, and an entertaining storyline. Get ready to immerse yourself in the world of Pepe as you navigate through exciting levels, vanquishing enemies, and ultimately facing the menacing Endboss!

Pokédex

JavaScript | HTML | CSS | Api

The "Pokédex" is a web project implemented using JavaScript, HTML, CSS, and API integration. Leveraging the Pokédex API library, the application dynamically renders Pokémon cards, allowing users to explore a comprehensive collection of Pokémon. Each card provides a snapshot of a Pokémon's essential information.

Users can delve deeper into the details of each Pokémon by accessing a dedicated detail view. This feature offers in-depth information, enabling users to learn more about the characteristics, abilities, and statistics of their favorite Pokémon.

The "Pokédex" project aims to provide an interactive and visually appealing platform for Pokémon enthusiasts to discover and explore the vast world of Pokémon. Whether you're a seasoned trainer or a newcomer to the Pokémon universe, this project offers a user-friendly experience for discovering and learning more about the fascinating creatures that inhabit the Pokémon world.
